# LIC Insurance Analytics Dashboard

## Project Overview

This project is an end-to-end Data Analytics solution built using Excel, MySQL, SQL, and Power BI. The dashboard helps analyze insurance business performance through interactive visualizations and key performance indicators.

## Tools Used

- Microsoft Excel
- MySQL
- SQL
- Power BI

## Dataset

The project includes the following datasets:

- Customers
- Agents
- Policies
- Policy Sales
- Claims

## Dashboard Features

- Total Customers
- Total Agents
- Total Premium
- Total Sales
- Total Claims
- Customer Distribution by City
- Customer Distribution by Gender
- Payment Mode Distribution
- Monthly Policy Sales
- Claim Status Analysis

## SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINS
- Aggregate Functions

## Business Insights

- Identified monthly sales trends.
- Compared claim status across policies.
- Analyzed customer demographics.
- Evaluated payment mode usage.
- Measured premium collection.
